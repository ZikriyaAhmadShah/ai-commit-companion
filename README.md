# ai-commit-companion


An intelligent CLI tool designed for open-source contributors that automatically generates structured, meaningful Git commit messages from a brief description of your changes using AI.

## Features
- **Instant Generation:** Uses an AI backend to turn messy thoughts into clean commits.
- **Open Source Focused:** Helps beginners adhere to conventional commit standards.
- **Zero Configuration:** No API keys required to test.

## Tech Stack
- Python 3.x
- Requests (API Integration)
